Series de Tiempo:
Conceptos y Código

Una guía visual que traduce la lógica matemática de los pronósticos a la sintaxis práctica de Python.

1. Preparación de Datos (Pandas)

El Concepto

Una serie de tiempo necesita que la computadora entienda que los datos están atados a una fecha específica y ordenada. Dividimos el tiempo en pasado lejano (Entrenamiento), pasado reciente (Validación) y futuro inmediato (Prueba).

# 1. Leer archivo saltando fila de error
df = pd.read_csv('datos.csv', skiprows=1)

# 2. Convertir texto a "Objeto Fecha"
df['mes'] = pd.to_datetime(df['mes'])

# 3. Hacer que la fecha sea la columna vertebral
df.set_index('mes', inplace=True)

# 4. Rebanar (Slicing) para entrenar y validar
train = df[:-24]

2. Medición del Error (Scikit-Learn)

El Concepto

Usamos el MAE (Error Absoluto Medio). Simplemente responde a la pregunta: "En promedio, ¿por cuántos puntos se equivocó mi predicción respecto a lo que realmente pasó?".

# Importar la herramienta de la librería estándar
from sklearn.metrics import mean_absolute_error

# Calcular el error matemático
error = mean_absolute_error(y_real, y_prediccion)

3. Modelo Ingenuo (Baseline)

El Concepto

El modelo del "sentido común". Asume que el patrón es un ciclo perfecto. Su lógica: "Las ventas de este diciembre serán exactamente iguales a las ventas del diciembre pasado".

# Extraer los últimos 12 meses de la tabla
# .iloc localiza por índice, .values saca los números
prediccion_ingenua = train['y'].iloc[-12:].values

4. Modelo ARIMA (Statsmodels)

El Concepto

Estadística pura. Evalúa cómo el pasado afecta al hoy (AutoRegresivo), estabiliza tendencias sumando/restando (Integrado) y suaviza el ruido (Media Automática). Con SARIMAX le añadimos ciclos de 12 meses.

from statsmodels.tsa.statespace.sarimax import SARIMAX

# Configurar componentes y ciclo de 12 meses
modelo = SARIMAX(train, order=(0,1,1), seasonal_order=(0,1,1,12))

# .fit() procesa los datos y aprende patrones
resultado = modelo.fit(disp=False)

# .forecast() predice el futuro
prediccion = resultado.forecast(steps=12)

6. Modelo LSTM (Deep Learning)

El Concepto

Una Red Neuronal con memoria. Recibe un bloque de 6 meses (contexto) para adivinar el 7mo mes. Aprende por ensayo y error (Épocas), ajustando sus neuronas en reversa (Backpropagation) cuando se equivoca.

Requiere escalar datos a 0-1 (MinMaxScaler) y usar Tensores para GPU.

La Gran Conclusión

A pesar de la popularidad de la Inteligencia Artificial (LSTM), para series de tiempo mensuales, cortas y con ciclos muy predecibles, la estadística clásica (ARIMA) suele ser más precisa y confiable. ¡Lo complejo no siempre es mejor!
